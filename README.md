## Hızlı Başlangıç

Gereksinimler:
- GCC veya herhangi bir C derleyici

Derleme:
```bash
gcc -o rescue Rescue-The-Baker.c
```

Çalıştırma:
```bash
./rescue

rescue.exe
```
